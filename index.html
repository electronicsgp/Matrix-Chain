Solución por fuerza bruta

Considerando, por ejemplo, el método de fuerza bruta que intenta calcular el número de multiplicaciones escalares de cada orden posible. Por ejemplo, si tenemos cuatro matrices M1, M2, M3 y M4, el algoritmo probará los siguientes cinco ordenamientos: (M1(M2(M3M4))), (M1((M2M3)M4)), ((M1M2)(M3M4)), ((M1M2)M3)M4)), ((M1(M2M3))M4). En general, el número de ordenaciones es igual al número de formas de colocar paréntesis para multiplicar las n matrices de todas las formas posibles. Sea f (n) el número de formas de entre paréntesis completo un producto de n matrices. Supongamos que queremos realizar la multiplicación: (M1M2 ...Mk) × (Mk+1Mk+2 ...Mn). Entonces, hay f (k) formas de poner entre paréntesis las primeras k matrices. Para cada una de las f (k) formas, hay f (n − k) formas de entre paréntesis las n - k matrices restantes, para un total de f (k) f (n - k) formas. Dado que k puede asumir cualquier valor entre 1 y n - 1, el número total de formas de poner entre paréntesis las n matrices viene dado por la suma: f(n)=∑_(k=1)^(n-1)▒〖f(k)f(n-k)〗 Observe que solo hay una forma de multiplicar dos matrices y dos formas de multiplicar tres matrices. Es decir, f (2) = 1 y f (3) = 2. Para que la recurrencia tenga sentido, dejamos f (1) = 1. Se puede demostrar que f(n)=1/n ((2n-2)¦(n-1)) Esta recurrencia genera los denominados números catalanes definidos por Cn = f (n + 1), los primeros diez términos de los cuales son 1, 1, 2, 5, 14, 42, 132, 429, 1430, 4862, 16796, .... Así, por ejemplo, hay 4862 formas de multiplicar diez matrices. Por la fórmula de Stirling n!≈ √2πn (n/e)^n,donde e=2.71828…, Tenemos f(n)=1/n ((2n-2)¦(n-1))=(2n-2)!/(n((n-1)!)^2 )≈4^n/(4√π n^1.5 ) Así, f(n)=Ω(4^n/n^1.5 ) Dado que, para cada expresión entre paréntesis, encontrar el número de multiplicaciones escalares cuesta Θ (n), se deduce que el tiempo de ejecución del método de fuerza bruta para encontrar la forma óptima de multiplicar las n matrices es Ω (4n / √n), que es impráctico incluso para valores de n de tamaño moderado.
